# Topic 03: Android Basics

## Summary
Core Android framework concepts that explain how an app is launched, displayed, configured, and restored.

## Subtopic Index
- Activity Lifecycle
- Fragment Lifecycle
- State Restoration
- Context
- Intents
- Manifest & Resources
- Permissions
- Legacy UI Fundamentals

## Mental Model
Android components are lifecycle-driven entry points managed by the OS.

## Common Mistakes
- Holding an Activity context longer than the Activity lifecycle.
- Assuming process death behaves like rotation.
- Requesting dangerous permissions without handling denial.
