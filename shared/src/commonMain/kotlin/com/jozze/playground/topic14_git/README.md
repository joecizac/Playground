# Topic 14: Git

## Summary
Version control commands, history management, and team workflows.

## Subtopic Index
- Core Commands
- Branching & Merging
- Merge vs Rebase
- Cherry-Pick
- Stash
- Reset
- Revert
- GitFlow vs Trunk-Based Development

## Mental Model
Git stores snapshots and named pointers; workflows are policies around that model.

## Common Mistakes
- Using `reset --hard` on shared or uncommitted work.
- Rebasing public branches without agreement.
- Confusing revert with reset.
