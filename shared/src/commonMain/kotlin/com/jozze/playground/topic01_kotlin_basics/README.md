## Mental Model
Prefer small, explicit examples that show the language feature and the generated/runtime behavior that matters.

## Common Mistakes
- Treating nullable and non-null types as documentation only.
- Forgetting that sequences are lazy and collections are eager.
- Using `var` when a stable `val` communicates intent better.
