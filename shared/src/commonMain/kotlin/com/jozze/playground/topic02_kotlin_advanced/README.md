# Topic 02: Kotlin Advanced

## Summary
Advanced Kotlin language features that make APIs expressive and safe.

## Subtopic Index
- Extensions
- Higher-Order Functions
- Lambdas & Anonymous Functions
- Scope Functions
- Delegation
- Sealed Classes & Interfaces
- Generics
- Inline & Reified
- Type Aliases
- Objects & Companion Objects
- Exception Handling

## Mental Model
These features are tools for modeling intent; use them when they reduce ambiguity or boilerplate.

## Common Mistakes
- Overusing scope functions until receiver/argument meaning becomes unclear.
- Assuming extension functions override members.
- Using `!!` instead of modeling absence explicitly.
