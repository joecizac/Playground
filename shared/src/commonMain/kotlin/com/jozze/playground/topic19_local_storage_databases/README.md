# Topic 19: Local Storage / Databases

Persisting app data with relational databases, key-value stores, and files.

## Subtopic Index

## Mental Model
Storage choice depends on structure, query needs, platform support, and migration cost.

## Common Mistakes
- Storing secrets in plain preferences.
- Skipping migration tests.
- Blocking the main thread with database or file work.
