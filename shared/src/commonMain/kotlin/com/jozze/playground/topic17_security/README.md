# Topic 17: Security

## Summary
Protecting data at rest, data in transit, secrets, and app code.

## Subtopic Index
- Android Keystore
- Encrypted Storage
- TLS
- Certificate Pinning
- Encryption Types
- Secret Management
- Obfuscation
- Biometrics

## Mental Model
Security is about reducing exposure and choosing protections that match the threat model.

## Common Mistakes
- Committing static secrets.
- Logging tokens or sensitive payloads.
- Treating obfuscation as encryption.
